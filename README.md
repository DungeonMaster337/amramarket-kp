## Ключевая позиция предложения (не терять при правках)

Клиент смешал три несовместимые вещи: «PWA» + «в оба стора» + «гарантия, что не удалят».
Честный разбор — это и есть отстройка от конкурентов:

- **100% гарантию не даёт никто** — решают Apple/Google. Обещать её = соврать.
- **Чистый PWA в сторы не попадает.** Тонкую webview-**обёртку Apple массово отклоняет** (Guideline 4.2).
- **Push на iOS-PWA** — только 16.4+, только с домашнего экрана, нестабильно. Для доски объявлений не годится.
- **Правильный путь:** Android → PWA/TWA ок; iOS → **настоящее нативное** (React Native/Expo, один код → оба стора).

## Что осталось выяснить у клиента (до сметы)

- **Есть ли у amramarket.com API / доступ к бэкенду?** Сайт похож на коробочную CMS.
  Если API нет — нужен API-слой, это заметная часть сметы. Не обещать бесшовную интеграцию заранее.
